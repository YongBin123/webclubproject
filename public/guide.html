여행지 추천 사이트 사용가이드

메인화면 페이지

  1. 이 사이트는 여행지 추천 정보를 알아볼 수 있으며, 각종 서비스를 이용할 수 있는 사이트입니다.
  2. 좌측에 각종 이용 가능 메뉴들이 있습니다. 원하시는 메뉴를 선택하시면 됩니다.
  3. 메뉴 아이콘을 클릭하면 자유롭게 메뉴를 보였다가 없앴다가 하실 수 있습니다.
  4. 지역별 여행지 추천을 누르면 아코디언 형식으로 지역이 쭉 나열되며, 다시 한 번 지역별 여행지 추천을 누르면 지역이 사라집니다.
  5. 좌측의 '고객센터 지원' 부분을 통해 문의사항을 문의할 수 있습니다.
  6. 좌측 하단의 '이용 안내'를 통해 사이트 이용에 대한 안내를 받으실 수 있습니다. 원하는 서비스를 클릭하시면 됩니다.
  7. 상단의 검색 부분에 주소를 알고 싶은 여행지의 이름을 쓰고 검색 버튼을 누르면 여행지의 주소를 확인하실 수 있습니다.
  8. 우측 상단에서 로그인을 할 수 있으며, 회원가입 버튼을 누르면 회원가입 절차를 진행하실 수 있습니다.
  9. 좌측 하단의 '이용 안내'를 통해 사이트 이용에 대한 안내를 받으실 수 있습니다. 원하는 서비스를 클릭하시면 됩니다.
  10. 하단에는 지역별로 가장 인기가 많은 1위 여행지에 대한 주소 정보와 사진을 보실 수 있습니다. 사진을 클릭하시면 홈페이지로 이동합니다.

지역별 여행지 추천 페이지

  1. 원하는 지역을 클릭합니다.
  2. 해당 지역의 추천 여행지 10곳의 주소와 사진 정보가 나옵니다.
  3. 관심 있는 여행지를 클릭하면 해당 여행지의 홈페이지로 연결됩니다.
  4. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  5. 이로써 추천 여행지들에 대한 정보들을 얻어갈 수 있습니다.

여행 퀴즈 이벤트 페이지

  1. 첫번째 문제가 뜨면 본인이 알고 있는 답을 적습니다.
  2. 정답을 확인하고 싶을 경우는 '정답 확인' 버튼을 클릭하면 정답 유무와 정답을 알려줍니다.
  3. 문제를 건너뛰고 싶거나 정답 확인 후 다음 문제로 넘어가고 싶으면 '다음 문제로' 버튼을 클릭하면 됩니다.
  4. 마지막 문제가 끝나게 되면 본인의 점수를 화면 아래에서 확인할 수 있습니다.
  5. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  6. 이로써 여행지 퀴즈 이벤트 페이지를 즐기실 수 있습니다.

여행 패킹 리스트 페이지

  1. 여행지의 유형과 원하는 계절을 선택합니다
  2. 아이템 보기 버튼을 클릭합니다.
  3. 선택한 것에 해당하는 필수 패킹 리스트가 화면에 뜨게 됩니다.
  4. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  5. 이로써 여행 패킹 리스트 페이지를 유용하게 사용하실 수 있습니다.

여행 메모 페이지

  1. 메모 작성란에 본인이 작성하고 싶은 메모를 작성합니다.
  2. 저장 버튼을 클릭합니다.
  3. 화면에서 본인이 방금 저장했던 메모를 보실 수 있습니다.
  4. 중간에 메모를 작성하다가 작성 중인 내용 전체를 지우고 싶으면 지우기 버튼을 클릭합니다.
  5. 저장된 메모를 삭제하고 싶으면 저장된 메모 옆에 있는 '삭제' 글자를 클릭하면 삭제됩니다.
  6. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  7. 이로써 여행 메모 페이지를 유용하게 사용하실 수 있습니다.

여행 플래너 페이지

  1. 본인이 계획하고 있는 여행 날짜, 장소, 예산, 교통, 숙박정보, 날씨, 일정 내용을 양식에 작성합니다.
  2. 저장 버튼을 클릭합니다.
  3. 화면에 본인이 입력한 내용이 뜨게 됩니다.
  4. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  5. 이로써 여행 플래너 페이지를 유용하게 사용하실 수 있습니다.

여행지 리뷰 페이지

  1. 본인이 리뷰를 작성하고 싶은 여행지 이름, 리뷰 내용을 작성하고 별점을 체크합니다.
  2. 리뷰 저장 버튼을 클릭합니다.
  3. 화면에서 본인이 방금 저장했던 리뷰를 '리뷰 목록'을 통해 보실 수 있습니다.
  4. 중간에 리뷰를 작성하다가 작성 중인 내용 전체를 지우고 싶으면 지우기 버튼을 클릭합니다.
  5. 저장된 리뷰를 삭제하고 싶으면 저장된 리뷰 옆에 있는 '삭제' 글자를 클릭하면 삭제됩니다.
  6. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  7. 이로써 여행지 리뷰 페이지를 유용하게 사용하실 수 있습니다.

여행 커뮤니티 페이지

  1. 본인의 이름, 글 제목, 글 내용을 작성합니다.
  2. 글 작성 버튼을 클릭합니다.
  3. 화면에서 본인이 방금 저장했던 글을 '글 목록'을 통해 보실 수 있습니다.
  4. 중간에 글을 작성하다가 작성 중인 내용 전체를 지우고 싶으면 지우기 버튼을 클릭합니다.
  5. 저장된 글을 삭제하고 싶으면 저장된 글 옆에 있는 '삭제' 글자를 클릭하면 삭제됩니다.
  6. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  7. 이로써 여행 커뮤니티 페이지를 유용하게 사용하실 수 있습니다.

회원가입 페이지

  1. 본인이 원하는 아이디, 비밀번호, 비밀번호 확인란, 이메일, 이름, 전화번호를 작성합니다.
  2. 가입하기 버튼을 누르면 회원가입이 완료됩니다.
  3. 메인화면으로 돌아가고 싶으면 좌측 상단에 있는 '메인화면으로' 글자를 클릭합니다.
  4. 이로써 회원가입 페이지를 유용하게 사용하실 수 있습니다.
